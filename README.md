# Sky便利ツール Step4 実取得版

## 使い方
1. このZIPを解凍
2. 中身をすべて `alismemo/sky-tools` に上書きアップロード
3. Actions → Update Sky Data → Run workflow
4. 成功後、`data/sky.json` の `status` が `ok` になるか確認

## URL
通常版: https://alismemo.github.io/sky-tools/
OBS版: https://alismemo.github.io/sky-tools/overlay.html

## 自動更新
GitHub Actionsが毎日 JST 16:10 / 17:10 / 18:10 に自動実行します。
手動更新は Actions → Update Sky Data → Run workflow です。
